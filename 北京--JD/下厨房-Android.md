职责：

1. 负责下厨房 Android 客户端的产品研发、维护、优化及扩展；
2. 配合其他部门完成需求细化、技术方案评估并制定解决方案；
3. 负责优化 App 性能并提高线上稳定性。

要求：

1. 对技术有热情，热爱新的挑战；
2. 熟悉 Java 语言，或者熟悉 Kotlin 语言，熟悉 Android SDK
3. 熟悉 ListView/RecyclerView 优化、RxJava、Glide、OkHttp等；
4. 熟悉常用设计模式、内存泄露及性能优化；
5. 有团队协作精神
